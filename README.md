# Otimizando o Sistema Bancário com Funções Python

<img align="right" height="200" style="border-radius:50px;" src="https://hermes.dio.me/tracks/648ef080-6c4b-4e54-bf72-34f62030f350.png">

#### Repositório criado para o desafio do bootcamp Python AI Backend Developer da DIO.


<p>A função deste desafio é otimizar o Sistema Bancário previamente desenvolvido com o uso de funções Python. O objetivo é aprimorar a estrutura e a eficiência do sistema, implementando as operações de depósito, saque e extrato em funções específicas. Você terá a chance de refatorar o código existente, dividindo-o em funções reutilizáveis, facilitando a manutenção e o entendimento do sistema como um todo. Prepare-se para aplicar conceitos avançados de programação e demonstrar sua habilidade em criar soluções mais elegantes e eficientes utilizando Python.</p>
